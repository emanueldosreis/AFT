- agent financial identity model;
- event taxonomy;
- decentralized/federated trust model;
- peer discovery model;
- blockchain anchoring profile;
- quantum-resistant cryptography profile;

## Status

Draft 0.1 for public review.

- Specification is early.

- PQC profile requires expert cryptographic review.
- Payment protocol mappings require ecosystem review.
- Legal, financial, and compliance implications require specialist review.
- Conformance testing is incomplete.
